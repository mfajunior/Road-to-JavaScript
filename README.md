## 🚀 Progresso do Aprendizado

Aqui estão os conceitos fundamentais que explorei até o momento:

- **Fundamentos:** Instalação e configuração de ambiente utilizando **Vite**, estrutura básica de um projeto React e entendimento do **JSX**.
- **Componentização:** Criação, importação e exportação de componentes modulares para melhor organização do código.
- **Estilização:** Aplicação de CSS para criar interfaces funcionais e organizadas.
- **Props & Eventos:**
    - Passagem de dados entre componentes (Pai para Filho).
    - Manipulação de eventos (botões, cliques e ações do usuário).
- **Boas Práticas de Dados:**
    - Validação de propriedades com **PropTypes** para garantir a segurança dos dados.
    - Definição de valores padrão (`defaultProps`) para evitar que a UI quebre em estados vazios.
- **Renderização Dinâmica:** Manipulação de listas e renderização de elementos de forma eficiente.
